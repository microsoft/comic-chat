Introducing

Microsoft Script Chat 1.0

Microsoft Script Chat has many features available. These features include embedding html tags within the chat window, to dynamically change a given frame by using push technology.

Note: These options are only available if you are the host within the given channel.

To embed html tags within the chat window, type an html tag within the text box and then click on Send Message. For example:
<img src=../images/rose.gif>
To push a new document to a given frame, type
"push example.htm"
within the text box and then click on Send Message.
