Additional features

Microsoft Script Chat 1.0

Other features within this example include an enhances members list. Place your mouse pointer over an image within the members list will return information about the user. In this case the information displayed is the nick and whether the user is a host, speaker, or a spectator.

Also the room and user list open a new instance of the browser to display the information. This allows you to customize how the information is displayed to the user.

As you can see Microsoft Script Chat is very customizable and can be designed in many different ways to fit your needs.
