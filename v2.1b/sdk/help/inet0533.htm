Microsoft Chat Protocol 2.0


The Microsoft® Chat Protocol Control 2.0 is an ActiveX™ control that allows you to participate in a chat, or conversation using text and data between two or more users connected to a chat server. The Chat Protocol Control 2.0 communicates with the chat server through either the IRC or IRCX protocols.
