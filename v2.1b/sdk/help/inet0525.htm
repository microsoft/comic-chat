Microsoft Chat Control 1.1


The Microsoft® Chat Control 1.1 is an ActiveX™ control that allows you to participate in a chat, or conversation using text and data between two or more users connected to a chat server. The Chat Control 1.1 communicates with the chat server through either the MIC or IRC protocols.
